# deno-scraper
## Start server
deno run -c tsconfig.json --allow-net --allow-env --allow-read ./src/app.ts
or
denon start

## Debug mode
deno run -c tsconfig.json --allow-net --allow-env --allow-read ./src/app.ts
or
denon debug

then access chrome://inspect on a chorme browser, click incpect button.
